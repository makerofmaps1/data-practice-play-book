# Core Python

A collection of flat, linear Python scripts that demonstrate core language features and practical patterns. Each script is designed to run top-to-bottom with clear section headers and example outputs.

## Scripts

- core_data_types.py
  - Strings, integers, floats, lists, dictionaries, and practical data manipulation.
- functions_basics.py
  - Function fundamentals, arguments, closures, decorators, and higher-order functions.
- functions_advanced.py
  - Advanced decorators, retry/caching/validation patterns, generators, and composition.
- oop_patterns.py
  - Classes, inheritance, polymorphism, encapsulation, and common design patterns.
- file_io.py
  - Text, CSV, JSON, pickle, and path handling with real-world workflows.
- math_and_statistics.py
  - NumPy arrays, linear algebra, descriptive stats, t-tests, and A/B testing patterns.

## Requirements

- Python 3.11+
- numpy
- scipy

## Run

From the repository root:

```bash
python 01_core_python/core_data_types.py
python 01_core_python/functions_basics.py
python 01_core_python/functions_advanced.py
python 01_core_python/oop_patterns.py
python 01_core_python/file_io.py
python 01_core_python/math_and_statistics.py
```
